## 💻 실행 방법

1. **원격 저장소 복제**

```bash
$ git clone https://github.com/do-good-morning/do-good-morning.git
```

2. **프로젝트 폴더로 이동 후 서버 폴더로 이동**

```bash
$ cd do-good-morning
$ cd back
```

3. **서버 실행**

```bash
$ export FLASK_APP=back
$ export FLASK_ENV=development
$ flask run
```

4. **클라이언트 폴더로 이동**

```bash
$ cd client
```

> 새 창 터미널 열기 후

5. **필요한 node_modules 설치**

```bash
$ npm install
```

6. **리액트(클라이언트) 앱 실행**

```bash
$ npm start
```
